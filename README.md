# 模糊 PID 激光器温控系统（升级版）

## 项目概述

激光器的输出波长和功率稳定性对温度高度敏感。本项目设计了一套**高精度、低超调**的嵌入式温度控制系统：

- **控制对象**：激光器工作温度（目标值 35°C）
- **执行机构**：TEC 半导体制冷片（加热/制冷双向驱动）
- **控制算法**：模糊自适应 PID，在线整定 Kp/Ki/Kd 参数
- **显示**：0.96 寸 OLED 实时显示温度等参数
- **通信**：串口输出温度数据至上位机分析

**适用场景：** 半导体激光器温控、光模块恒温、精密实验温控平台。

## 模糊 PID 控制器

### 控制结构

本项目采用**模糊自适应 PID** 结构：以经典 PID 为基座，根据实时误差 e 和误差变化率 ec 在线调整 Kp / Ki / Kd 系数。

```
KP = kp0 + ΔKp(e, ec)
KI = ki0 + ΔKi(e, ec)
KD = kd0 + ΔKd(e, ec)
```

### 基础参数

| 参数 | 值 | 说明 |
|------|-----|------|
| kp0 | 50.0 | 比例系数基数 |
| ki0 | 0.5 | 积分系数基数（微弱，缓慢消除稳态误差） |
| kd0 | 50.0 | 微分系数基数 |
| 目标温度 | 35.0 °C | 默认设定值 |
| 控制周期 | 50 ms | 20 Hz 更新率 |

### 模糊化

- **输入论域：** e ∈ [-10, 10] °C，ec ∈ [-0.5, 0.5] °C/s
- **量化输出：** [-3, +3] 共 7 级
- **模糊集合：** {NB, NM, NS, ZO, PS, PM, PB}
- **隶属度函数：** 三角形线性插值，相邻集合重叠

### 规则表

#### Kp 规则表

| e \\ ec | NB | NM | NS | ZO | PS | PM | PB |
|---------|----|----|----|----|----|----|----|
| NB | PB | PB | PB | PB | PB | PB | PB |
| NM | PM | PM | PM | PM | PM | PM | PM |
| NS | PS | PS | PS | PS | PS | PS | PS |
| ZO | ZO | ZO | ZO | ZO | ZO | ZO | ZO |
| PS | PS | PS | PS | PS | PS | PS | PS |
| PM | PM | PM | PM | PM | PM | PM | PM |
| PB | PB | PB | PB | PB | PB | PB | PB |

> 规则解读：误差为正（温度偏低）时增加 Kp 加强加热，误差为负（温度偏高）时增加 Kp 加强制冷。零误差区 Kp 不调整，保持基值。

#### Ki 规则表

Ki 规则与 Kp 对称但方向不同，用于消除稳态误差。规则表经过优化确保零误差区无积分累积。

#### Kd 规则表

Kd 在当前配置下保持零调整（基值 50），微分作用通过基值提供，模糊层暂不干预。

### 解模糊

采用**加权平均法**：

```
ΔKp = Σ(隶属度 × 规则值) / Σ(隶属度)
```

查询相邻 4 条规则（e 的 2 个隶属度 × ec 的 2 个隶属度），加权求和后通过逆量化函数映射到实际调整量范围：

| 调整量 | 范围 | 说明 |
|--------|------|------|
| ΔKp | [-40, +40] | Kp 在线调整范围 |
| ΔKi | [-0.05, +0.05] | Ki 微调范围 |
| ΔKd | [-400, +400] | Kd 调整范围 |

### 关键保护机制

#### 1. 静默区（Dead Zone）

当误差 |e| < 0.3 °C 时，PID 参数退回基值，模糊层不干预，避免零位附近抖动：

```c
if (error > -0.3 && error < 0.3) {
    kp = kp0;  ki = ki0;  kd = kd0;
}
```

#### 2. 积分分离

当反馈温度低于目标值的 80% 时（刚启动阶段），**积分项完全关闭**，仅 PD 控制快速逼近目标：

```c
if (feedback >= 0.8 * setpoint) {
    // 启用积分
} else {
    integral = 0;  // 积分分离
}
```

#### 3. 抗积分饱和（Anti-Windup）

积分项在输出未饱和时才累积；积分值限幅在 ±2000 防止 Windup。

#### 4. 系数非负约束

```c
if (kp < 0) kp = 0;
if (ki < 0) ki = 0;
if (kd < 0) kd = 0;
```

## 温度采集与滤波

### 信号链路

```
NTC 热敏电阻 → 分压电路 → PA1 (ADC1_CH1) → DMA 连续转换 → 滑动均值滤波 → 温度换算
```

### ADC 配置

| 参数 | 值 |
|------|-----|
| 分辨率 | 12-bit (0-4095) |
| 参考电压 | 3.3V |
| 采样时间 | 28.5 个周期 |
| 转换模式 | 连续转换 + DMA 循环 |
| 触发方式 | 软件触发 |

### 滑动均值滤波

窗口大小 10 的环形缓冲区均值滤波，有效抑制 ADC 噪声：

```c
#define WINDOW_SIZE 10
float sliding_mean_filter(float new_data);
### 温度换算
```

AD 值 → 电压 → 热敏电阻阻值 → 温度的两段指数拟合曲线：

```c
Voltage = AD_Value / 4095 * 3.3;
RTH = 55.44 / (0.833 * Voltage + 2.31) - 12;
feedback = 90.93 - 56.38 * (1 - exp(-RTH/19.6344))
                 - 45.04 * (1 - exp(-RTH/2.986));
```

> 温度-电阻关系通过实测数据拟合得到，适用于该 NTC 热敏电阻和分压电路。

## TEC 驱动逻辑

PWM 占空比输出与 TEC 方向控制分离：

- **output >= 0（加热模式）：** PB5=1, PB6=0, PWM = output
- **output < 0（制冷模式）：** PB5=0, PB6=1, PWM = -output
- **最大占空比限幅：** PWM ≤ 1000

PWM 配置：

| 参数 | 值 |
|------|-----|
| 定时器 | TIM2_CH1 (PA0) |
| 频率 | 1 kHz (72 MHz / 72 / 1000) |
| 分辨率 | 1000 级 (0-1000) |

## 硬件连接

### 核心连接

| STM32F103C8 | 外设 |
|-------------|------|
| PA0 | TEC PWM 输入 |
| PA1 | NTC 热敏电阻分压输出 |
| PA9 | USB-TTL (TX) → 上位机 RX |
| PB5 | TEC 加热使能 (高有效) |
| PB6 | TEC 制冷使能 (高有效) |
| PB8 | OLED SCL |
| PB9 | OLED SDA |

### TEC 驱动电路（参考）

```
PB5 ──┬── 驱动 MOSFET (加热)
       │
PB6 ──┬── 驱动 MOSFET (制冷)
       │
PA0 ──┴── TEC 功率 PWM (两路共用)
```

> **注意：** TEC 驱动需要外部 MOSFET H 桥或半桥电路，PB5/PB6 控制方向，PA0 PWM 控制功率。禁止 PB5 和 PB6 同时为高。

## 构建与烧录

### 前提条件

安装 [Keil MDK-ARM](https://www2.keil.com/mdk5) 并安装 STM32F1 器件支持包 (DFP)。

### 步骤

1. 双击打开 Project.uvprojx（Keil 项目文件）
2. 在工具栏选择目标器件：**STM32F103C8**
3. 点击 **Build (F7)** 编译项目
4. 连接 ST-Link / J-Link 调试器
5. 点击 **Load (F8)** 烧录固件

### 串口监视

烧录后通过串口工具（如 SSCOM、PuTTY）连接 PA9（TX），波特率 115200：

```
35.123
35.089
35.041
...
```

> 串口每次输出一行当前温度值（°C），可用 Python / MATLAB 等工具实时绘制温度曲线。
